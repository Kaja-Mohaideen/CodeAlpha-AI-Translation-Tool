# 🌍 AI Translation Hub

A web application built with **Flask** that allows users to instantly convert text into multiple languages using the Deep Translator library.

## ✨ Key Highlights
- Detects source language automatically
- Supports translation across 100+ languages
- Copy results directly to clipboard
- Listen to translations with built‑in speech synthesis
- Clean interface with a modern banner design
